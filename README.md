# Random Password Generator

This project is a Random Password Generator 
designed to help users create strong and secure passwords with ease. 
It allows customization of password length and character types, 
ensuring both security and flexibility for different use cases.
## Features
- Generate random passwords instantly
- Customizable password length
- Option to include/exclude uppercase, numbers, and symbols
- Simple and easy-to-use interface

## Tech Stack
- HTML, CSS, JavaScript

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Set your desired password length and options
4. Click "Generate" to get your random password
